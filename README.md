# DSA Telemetry Judge

A self-hosted platform that tracks your problem-solving behavior and provides insights to improve your coding skills.

## Features (Planned)
- Real-time telemetry tracking (keystrokes, timing, undo rate)
- Problem management system
- Performance analytics dashboard
- "Tilt meter" for detecting frustration
- ML-based attempt prediction

## Tech Stack
- **Backend**: FastAPI, PostgreSQL, Redis
- **Frontend**: React/Next.js
- **Deployment**: Docker, CI/CD

## Project Status
🚧 Phase 0: Project Setup - IN PROGRESS

## Development

### Prerequisites
- Python 3.9+
- Node.js 18+
- PostgreSQL 14+
- Redis 6+

### Setup
```bash
# Clone repository
git clone <your-repo-url>
cd dsa-telemetry-judge

# Backend setup
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Frontend setup
cd ../frontend
npm install
```
